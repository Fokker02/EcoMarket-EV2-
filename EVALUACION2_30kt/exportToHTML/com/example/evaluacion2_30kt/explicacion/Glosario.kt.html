11   // ==========// Variables// ==========

13   /**
14    * DarkColorScheme:
15    * Es una variable que guarda la paleta de colores específica para el TEMA OSCURO de tu aplicación.
16    * Define colores como el primario, secundario, fondo, etc., que se usarán cuando el modo oscuro esté activo.
17    */
18   
19   /**
20    * LightColorScheme:
21    * Similar a la anterior, pero esta variable guarda la paleta de colores para el TEMA CLARO.
22    * Se usará cuando la aplicación no esté en modo oscuro.
23    */
24   
25   
26   // ==========// Funciones// ==========
27   
28   /**
29    * darkColorScheme(...) y lightColorScheme(...):
30    * Son funciones que te da Material Design 3 para CREAR las paletas de colores.
31    * Dentro de ellas, asignas un color específico a un rol (por ejemplo, `primary = Purple40`).
32    */
33   
34   /**
35    * @Composable fun EVALUACION2_30ktTheme(...):
36    * Esta es la FUNCIÓN PRINCIPAL de tu tema. Es un "Composable" que envuelve toda tu aplicación
37    * (o una parte de ella) para aplicarle un estilo consistente (colores y fuentes).
38    */
39   
40   /**
41    * isSystemInDarkTheme():
42    * Es una función que DETECTA si el sistema operativo del teléfono está en MODO OSCURO.
43    * Se usa para que tu aplicación cambie de tema automáticamente junto con el teléfono.
44    */
45   /** revisar los archivos despues de cada repaso, no es necesario memorizarlo
46    */
47   /**
48    * dynamicDarkColorScheme(...) y dynamicLightColorScheme(...):
49    * Estas funciones son para el COLOR DINÁMICO (Material You) en Android 12 y superior.
50    * Crean una paleta de colores automáticamente basándose en el fondo de pantalla del usuario.
51    */
52   
53   /**
54    * SideEffect { ... }:
55    * Es un tipo especial de Composable que se usa para ejecutar código que no está directamente relacionado con la UI,
56    * como cambiar el color de la barra de estado del sistema. Se ejecuta después de que la interfaz se haya dibujado.
57    */
58   
59   /**
60    * MaterialTheme(...):
61    * Es el CORAZÓN DEL SISTEMA DE TEMAS en Jetpack Compose. Esta función toma el `colorScheme` (la paleta de colores),
62    * `typography` (los estilos de texto) y los aplica a todos los elementos de la interfaz de usuario que contiene.
63    */
